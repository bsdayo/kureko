# Kureko「暮子」

极简主义的个人博客平台。

[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new/clone?repository-url=https%3A%2F%2Fgithub.com%2Fbsdayo%2Fkureko&env=NUXT_PUBLIC_POCKETBASE_URL)

## 状态

WIP

## 技术栈

- Nuxt.js 3 + Vue.js 3
- UnoCSS (`preset-wind4` + `preset-typography`)
- 基于 [Reka UI](https://github.com/unovue/reka-ui) 的界面组件，衍生自 [shadcn-vue](https://github.com/unovue/shadcn-vue/)
- [PocketBase](https://github.com/pocketbase/pocketbase/)
